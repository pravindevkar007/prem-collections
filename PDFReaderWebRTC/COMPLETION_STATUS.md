## 🚀 How to Use

### Quick Start
1. **Configure API Key**: Add NVIDIA API key to `backend/.env`
2. **Run Application**: Execute `start.bat` (Windows) or manual startup
3. **Access Frontend**: Open http://localhost:3000
4. **Upload PDF**: Drag & drop or browse to select PDF
5. **Interact**: Use voice or text to ask questions

### Features Available
- 📄 **PDF Processing**: Upload and extract text from PDFs
- 🎤 **Voice Input**: Click microphone to speak questions
- 🔊 **Voice Output**: AI responses read aloud automatically
- 💬 **Chat History**: View conversation with timestamps
- 🧠 **AI Summary**: Automatic document summarization
- 🔄 **Session Reset**: Clear and start fresh anytime
