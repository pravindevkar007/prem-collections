# Prem Collections - Frontend

Premium Fashion Store built with React.js

## 🚀 Live Demo
[Visit Prem Collections](https://prem-collections-client.netlify.app)

## 🛠️ Tech Stack
- React 19
- Tailwind CSS
- Framer Motion
- React Router DOM
- Axios
- Lucide React Icons

## 📦 Installation

```bash
npm install
npm start
```

## 🏗️ Build

```bash
npm run build
```

## 🌐 Deployment
This app is deployed on Netlify with automatic deployments from the main branch.

## 🔗 Backend API
Backend Repository: [prem-collections-server](https://github.com/Pravindevkar007/prem-collections-server)

## 📱 Features
- Responsive design
- Shopping cart functionality
- User authentication
- Admin panel
- Product management
- Order tracking
